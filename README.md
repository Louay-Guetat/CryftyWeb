## To run the project you need to run these commands: 
- composer update
- composer install 
- yarn install
- yarn add --dev @symfony/webpack-encore
- yarn add webpack-notifier --dev
- yarn encore dev

### NB: yarn v1.22.17 / composer v2.2.4 / php v7.4.27 / symfony4
